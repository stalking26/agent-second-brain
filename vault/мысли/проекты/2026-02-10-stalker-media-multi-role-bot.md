## Context
Голосовая запись 07:35 — масштабная концепция мультиролевого бота для Сталкер Медиа.

## Концепция
Бот-ассистент с ролевым доступом для команды компании:

### Роли и функции
1. **Наталья Космач (директор/жена)** — видит отчёты по компании, стратегию, финансы, может загружать Excel с финансовыми данными. AI анализирует и даёт рекомендации.
2. **Ольга Рижикова (помощник руководителя)** — отчитывается о проблемах, получает инструкции по решению. Данные по продавцам, CRM, финансам.
3. **Маркетолог** — получает рекомендации по лендингам, постингу, статистику. Отчитывается о прогрессе.
4. **Рома (HR/рекрутинг)** — отчёты по собеседованиям, наём для других компаний, рекомендации по автоматизации.

### Система отчётов
- Каждый сотрудник получает утренний отчёт в Telegram
- Вечером заполняет форму "что сделано"
- AI анализирует эффективность каждого → приватный отчёт руководителю
- Директор видит эффективность маркетолога, продавцов, помощника

### Onboarding-система (будущее)
- Автоматическая подготовка новых сотрудников через Telegram
- Обучающие материалы, проверка понимания
- Цель: "функция выполняется, переменные слагаемые — сумма не меняется" (быстрая замена специалистов)
- Потенциальный SaaS-продукт для клиентов (малый бизнес)

## Implication
Это расширение концепции admin panel и ERP — теперь с полноценным мультиролевым доступом. Можно обкатать на Сталкер Медиа и продавать как SaaS.

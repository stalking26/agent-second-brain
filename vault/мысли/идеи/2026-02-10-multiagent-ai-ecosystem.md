## Context
Запись 17:22 — концепция мультиагентной AI-экосистемы. Андрей рассказал о боте в группе и описал видение: не один бот, а армия специализированных ИИ, каждый — лучший в своей задаче.

## Концепция: Мультиагентная экосистема

Центральный AI-ассистент (Брайан) + специализированные ИИ-боты, каждый оптимизирован по соотношению качество/цена для своей задачи.

### Специализированные боты

| Бот | Задачи |
|-----|--------|
| **Генератор контента** | Картинки, презентации, видео |
| **Исследователь** | Поиск в интернете, мониторинг информации, трекинг |
| **Архивариус** | Составление продуманной базы данных для других ботов |
| **Финансист** | Учёт финансов, объяснение "что/почему/зачем", альтернативы |
| **Маркетолог** | Стратегия продвижения, аналитика, контент-план |
| **Веб-мастер** | Лендинги, сайты, приложения для разных программ |
| **Администратор** | Контроль правил, уведомления, проверка понимания, эскалация |

### Бот-Администратор (детально)
- Следит за выполнением правил всеми участниками
- При новом правиле: уведомление → вопросы на понимание → проверка последствий
- Эскалация: мягкое напоминание → меры согласно роли/должности/типу сотрудничества
- Адаптивный подход: учитывает роль и формат сотрудничества

### Принцип
Каждый ИИ выбирается по критерию: "кто решает задачу лучше всего и дешевле". Не один дорогой универсал, а экосистема оптимальных специалистов.

## Implication
Это эволюция multi-role bot (Stalker Media) в полноценную **AI-оркестрацию**. Брайан из ассистента становится координатором армии ботов. Потенциальный SaaS-продукт для малого бизнеса.

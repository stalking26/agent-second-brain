## Context
Рабочее Место Эксперта (Consultant Workbench) — модуль Agent Second Brain для проведения консультаций с "послойной" демонстрацией доказательств. Оправдывает чек 500+ BYN: клиент платит не за "гадалку", а за "аудит с доказательствами".

## 1. Визуальная среда (Consulting Board)
- **Технология:** Obsidian Canvas / Excalidraw
- **Функции:** Лазерная указка, графики, схемы
- **Сценарий:** Демонстрация логики принятия решений в реальном времени (Zoom/Meeting)

## 2. Работа с документами (Evidence Base)
- PDF-слои: исходный документ остаётся чистым
  - **Слой 1:** Комментарии эксперта
  - **Слой 2:** Маркеры/хайлайты (акценты)
- **Методика:** "Доказательная база" — клиент видит связь: факт → вывод эксперта

## 3. Ретроспективный анализ (AI Laboratory)
- Фиксация истории прогнозов и оценок
- Работа над ошибками: возврат к старым кейсам
- База для дообучения нейросетей в будущем

## 4. Медиа с таймкодами (Video/Audio)
### Вариант А: Плагин "Media Extended" (рекомендуется)
- Вставка видео (локальное / YouTube)
- Таймкоды в формате `01:30` или `10:05` — клик перематывает
- Фрагменты: `video.mp4#t=10,20` — воспроизведение только 10-20 сек

### Вариант Б: Стандартный HTML (без плагинов)
- `<audio src="запись.mp3#t=15,45" controls></audio>`

### Сценарий использования
Заметка "Сценарий встречи с клиентом Х":
- `[Интонация кандидата при вопросе о зарплате (04:12)]`
- `[Момент неуверенности при ответе про опыт (08:30)]`

## Связь с монетизацией
- Послойная демонстрация → обоснование чека 500+ BYN
- WebApp (TWA) + Consulting Board = визуальная убедительность
- Ретроспективный анализ → "история успеха" для маркетинга

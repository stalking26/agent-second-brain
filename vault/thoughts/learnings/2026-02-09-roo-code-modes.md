# Roo Code: Modes (Режимы) — шпаргалка

MCP = «руки» (инструменты), Modes = «шляпы» (роли).

## Доступные режимы

| Mode | Назначение |
|------|-----------|
| **Merge Resolver** | Решение Git-конфликтов автоматически |
| **Documentation Writer** | Структурированные инструкции и методички |
| **Project Research** | Глубокий анализ чужого кода/библиотек |
| **Coding Teacher** | Объяснение кода простыми словами |
| **Security Reviewer** | Аудит безопасности перед продажей бота |
| **DevOps** | Настройка серверов, Docker, CI/CD |
| **User Story Creator** | Перевод идей в чёткое ТЗ |

## Рекомендовано к установке (приоритет)

1. **Merge Resolver** — решение конфликтов Git
2. **Documentation Writer** — тренинги и заметки Obsidian
3. **Coding Teacher** — обучение при администрировании сервера

## Установка

Вкладка Modes → синяя кнопка Install. Ключи не нужны.

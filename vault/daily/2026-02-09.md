Вот четкая инструкция без воды. Мы договорились: ставим метод NPX, область Global (чтобы работало во всех проектах).

Ниже список того, что нужно ставить, где брать ключи и сколько это стоит.

1. Brave Search (Глаза в интернет)
Самый важный навык. Позволяет Клоду гуглить зарплаты, конкурентов и новости 2026 года.

Ссылка на ключ: api.search.brave.com

Тариф: Free (Бесплатно).

Лимит: 2,000 запросов в месяц (этого хватит за глаза для стратегии).

Карта: Может попросить привязать, но деньги не спишет, пока сами не включите платный тариф.

Куда вставлять: В поле BRAVE_API_KEY при установке.

2. Sequential Thinking (Мозг Архитектора)
Нужен для сложных задач. Позволяет Клоду сначала подумать, построить план, а потом писать код.

Ссылка на ключ: Не нужна.

Тариф: Бесплатно (работает за счет токенов самого Клода).

Установка: Просто найдите в списке MCP и нажмите Install. Ключ вводить не надо.

3. Memory (Долгосрочная память)
Позволяет Клоду запомнить вашу цель "5000 BYN" и структуру проекта навсегда.

Ссылка на ключ: Не нужна.

Тариф: Бесплатно (хранит данные у вас в папке).

Установка: Просто нажмите Install.

4. Todoist (Руки для задач)
Если вы хотите, чтобы Клод сам создавал вам задачи ("Позвонить кандидату", "Проверить сервер").

Ссылка на ключ: todoist.com/app/settings/integrations/developer

Где искать: Зайдите в "Integrations" -> "Developer" -> Скопируйте API token.

Тариф: Free (Бесплатно для личного использования).

Куда вставлять: В поле TODOIST_API_TOKEN.
